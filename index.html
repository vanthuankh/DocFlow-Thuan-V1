Biểu mẫu (Templates)

.docx
Với file Word (.docx):
  1. Thiết kế mẫu tài liệu như bình thường trong Word
  2. Tại vị trí cần điền dữ liệu, gõ {{TEN_BIEN}}
  3. Lưu file dạng .docx
Quy tắc đặt tên (tự động nhận dạng):
  • DATE_ → Chọn ngày (VD: DATE_NGAY_KY)
  • NUM_ → Nhập số/tiền (VD: NUM_GIA_TRI)
  • AREA_ → Văn bản dài (VD: AREA_GHI_CHU)
  • SELECT_ → Danh sách chọn
  • TABLE_ → Bảng động (VD: TABLE_HANG_HOA)
  • SIGN_ → Tên người ký
  • (khác) → Ô nhập văn bản thường
Các biến tự động:
  • {{TONG_CONG}} → Tổng tiền
  • {{TIEN_BANG_CHU}} → Số tiền bằng chữ
  • {{NGAY_THANG_NAM}} → Ngày hiện tại
  • {{MA_TAI_LIEU}} → Số tài liệu tự sinh
  • Chưa có biểu mẫu nào.

Cài đặt Google API

Kết nối với Google Drive và Google Sheets của bạn.

Cách lấy thông tin xác thực:

  1. Vào Google Cloud Console → tạo Project mới
  2. Enable Google Drive API và Google Sheets API
  3. Vào APIs & Services → Credentials
  4. Tạo OAuth 2.0 Client ID (loại Web application) → copy Client ID
  5. Tạo API Key → copy API Key
  6. Thêm domain của bạn vào Authorized JavaScript origins

Cần enable Drive API + Sheets API và restrict API Key cho 2 API này.

Lấy từ URL: https://docs.google.com/spreadsheets/d/[SHEET_ID]/edit

⚠ Sheet phải có 2 tab: "Khách Hàng" và "Người Liên Hệ" (dùng file mẫu đã tải).
