Biểu mẫu (Templates)

.docx
Với file Word (.docx):
  1. Thiết kế mẫu tài liệu như bình thường trong Word
  2. Tại vị trí cần điền dữ liệu, gõ {{TEN_BIEN}}
  3. Lưu file dạng .docx
Quy tắc đặt tên (tự động nhận dạng):
  • DATE_ → Chọn ngày (VD: DATE_NGAY_KY)
  • NUM_ → Nhập số/tiền (VD: NUM_GIA_TRI)
  • AREA_ → Văn bản dài (VD: AREA_GHI_CHU)
  • SELECT_ → Danh sách chọn
  • TABLE_ → Bảng động (VD: TABLE_HANG_HOA)
  • SIGN_ → Tên người ký
  • (khác) → Ô nhập văn bản thường
Các biến tự động:
  • {{TONG_CONG}} → Tổng tiền
  • {{TIEN_BANG_CHU}} → Số tiền bằng chữ
  • {{NGAY_THANG_NAM}} → Ngày hiện tại
  • {{MA_TAI_LIEU}} → Số tài liệu tự sinh
  • Chưa có biểu mẫu nào.

Cấu hình Google API

Lưu ý quan trọng:

  • Đảm bảo đã bật Google Drive API và Google Sheets API trong Google Cloud Console.
  • Thêm ${window.location.origin} vào danh sách "Authorized JavaScript origins" trong thông tin xác thực OAuth.
  • Biểu mẫu (Template) PHẢI là file .docx hoặc .xlsx. Hệ thống không đọc được trực tiếp Google Docs/Sheets.

Hệ thống sẽ tự động trích xuất ID nếu bạn dán toàn bộ đường dẫn URL.

Quản lý biểu mẫu

Yêu cầu về định dạng:

Hệ thống chỉ hỗ trợ file .docx (Word) hoặc .xlsx (Excel). Nếu bạn đang dùng Google Docs, hãy vào Tệp > Tải xuống > Microsoft Word (.docx) sau đó tải file đó lên thư mục DocFlow/_template trên Drive.
